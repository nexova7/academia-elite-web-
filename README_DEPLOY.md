# Instrucciones de Despliegue Élite

Para subir los cambios a la nube, ejecuta los siguientes comandos en tu terminal desde la carpeta `c:\Users\visitante\Desktop\academia-app`:

```bash
# 1. Preparar archivos
git add .

# 2. Confirmar cambios
git commit -m "feat: implementacion de panel de bienvenida, perfil estilo licencia y refactorizacion de seguridad"

# 3. Subir a la nube
git push origin main
```

> [!IMPORTANT]
> Los nuevos avatares se encuentran en la carpeta de la IA. Por favor, asegúrate de haber copiado los archivos `.png` a `project/frontend/` antes de realizar el commit.
