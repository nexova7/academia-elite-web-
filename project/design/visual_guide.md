# Visual Guidelines - Academia Élite "Prestige"

## 1. Color Usage
| Element | Color | Hex | Context |
| :--- | :--- | :--- | :--- |
| **Background** | Rich Black | `#0A0A0A` | Global background. Never use `#000000`. |
| **Cards/Surfaces** | Charcoal Grey | `#1F1F1F` | Content containers. |
| **Primary Actions** | Elite Red | `#C41E3A` | Buttons, Progress Bars, Active States. |
| **Accents/Icons** | Prestige Gold | `#D4AF37` | Borders, Icons, Text Highlights. |
| **Text** | Pure White | `#FFFFFF` | Headings (95% opacity), Body (85% opacity). |

## 2. Typography Hierarchy
*   **H1 (Hero)**: `Playfair Display`, Bold (700), 3.5rem. Uppercase.
*   **H2 (Section)**: `Playfair Display`, Bold (700), 2.5rem.
*   **H3 (Card Title)**: `Playfair Display`, Regular (400), 1.5rem.
*   **Body**: `Inter`, Light (300) for descriptions, Regular (400) for reading.

## 3. UI Elements

### Buttons
*   **Primary**: Solid Elite Red with a linear-gradient shine effect on hover.
*   **Secondary**: Transparent with Gold border. Gold glow on hover.

### Effects
*   **Glassmorphism**: Used on Navbar (`backdrop-filter: blur(10px)`).
*   **Glows**: Red glow (`box-shadow: 0 0 20px ...`) for primary actions. Gold glow for secondary.

## 4. Dos and Don'ts
*   ✅ **Do**: Use plenty of negative space.
*   ✅ **Do**: Use uppercase for short headers and buttons.
*   ❌ **Don't**: Use gradients on text (keep it clean white or gold).
*   ❌ **Don't**: Overuse the red glow; keep it for the main CTA only.
